### 🧠 About me

I'm a **Systems & Computer Engineer** from Medellín, Colombia, specializing in AI agents and autonomous systems. Currently building the AI Doctor at **[Telepatia AI](https://telepatia.ai)** (Stanford-born healthtech) and Founding Engineer at **Cerouno**.

I created **[agent-os](https://github.com/MiguelWO/agent-os)** — a minimal Python framework for building composable multi-agent pipelines using the **Plan → Execute → Critique** pattern on top of LangGraph.

Previously: Data Scientist at **Sofka Technologies** (multi-agent systems, LangGraph, RAG) · Engineering Intern at **BMW Group** in Munich · Exchange at **UPV Valencia**.

🗣️ `EN` · `ES` · `DE`
